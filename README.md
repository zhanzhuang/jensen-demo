# 表格

| 表头1 | 表头2 | 表头3 |
| --- |:---:| ---:|
| 内容默认居左 | 内容居中 | 内容居右 |
| 1 | 1 | 1 |

第二行分割表头和内容。
- 有一个就行，为了对齐，多加了几个
文字默认居左
-两边加：表示文字居中
-右边加：表示文字居右
注：原生的语法两边都要用 | 包起来。此处省略
